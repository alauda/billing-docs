# Введение

В эпоху стремительного развития облачных вычислений предприятиям необходима гибкая и точная модель выставления счетов для управления и оптимизации расходов на облачные ресурсы. Платформа предлагает продукт для выставления счетов, специально разработанный для облачно-ориентированных сценариев, который помогает вам достичь детального управления ресурсами и контроля затрат.

Платформа предоставляет обобщенные возможности для измерения и выставления счетов, а также основные методы выставления счетов, поддерживая измерение и выставление счетов за ресурсы CPU, памяти и хранения в кластерах, а также предлагая статистику и возможности подробного просмотра.

- Статистика затрат: поддерживает администраторов платформы в обобщении затрат для каждого арендатора.
- Подробности о затратах: позволяет администраторам платформы запрашивать подробности о затратах для каждого арендатора с целью сверки.

## Преимущества

- **Тщательное управление ресурсами**: точно измеряет использование ресурсов CPU, памяти и хранения, помогая пользователям глубже понять потребление ресурсов.
- **Контроль затрат**: с помощью детализированных данных измерений и информации о выставлении счетов пользователи могут лучше контролировать и оптимизировать затраты на ресурсы.
- **Гибкие методы выставления счетов**: поддерживает различные основные методы выставления счетов для удовлетворения разных потребностей пользователей.
- **Прозрачность**: предоставляет статистику использования ресурсов и функции детального просмотра, повышая прозрачность использования ресурсов.
- **Интеграция с OpenCost**: основана на OpenCost, что обеспечивает точность и надежность данных измерений.

## Сценарии применения

- **Управление ресурсами предприятия**: помогает предприятиям точно измерять и распределять затраты на IT-ресурсы, оптимизируя распределение ресурсов.
- **Облачно-ориентированные приложения**: подходят для архитектур микросервисов в облачно-ориентированных средах, точно измеряя потребление ресурсов каждым сервисом.
- **Многоарендные среды**: в многоарендных средах предоставляет независимое измерение и выставление счетов за ресурсы для каждого арендатора, обеспечивая справедливость и прозрачность.
- **Оптимизация затрат**: на основе детализированных данных о потреблении ресурсов определяет и оптимизирует ресурсы с высокими затратами для снижения общих операционных расходов.

## Ограничения использования

- **Зависимость от Prometheus**: данные измерений получаются на основе Prometheus; пожалуйста, убедитесь, что кластеры, требующие измерений, установили ACP Monitor с плагином Prometheus перед установкой.
- **Ограничения по типам ресурсов**: в настоящее время поддерживаются только измерение и выставление счетов за ресурсы CPU, памяти и хранения; другие типы ресурсов в данный момент не поддерживаются.
- **Совместимость кластеров**: функции могут быть несовместимы со всеми типами кластеров Kubernetes и должны использоваться в поддерживаемых средах кластеров.
- **Задержка данных**: из-за задержки в канале сбора данных возможно некоторое отклонение во времени получения данных измерений.

Исходя из этой функциональности, мы надеемся помочь пользователям лучше управлять и оптимизировать использование ресурсов своей контейнерной платформы, достигая более эффективного контроля затрат.
